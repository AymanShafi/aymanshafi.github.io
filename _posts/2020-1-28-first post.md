Brief description of why Jekyll and Github-Pages form a great combo for a static blog site free-of-cost.

![_config.yml]({{ site.baseurl }}/images/jekyll-logo-light-solid.png)

[Jekyll](https://jekyllrb.com/ "Jekyll") is a simple static site generator written in Ruby by Tom Preston-Werner, GitHub's co-founder.
## Few Advantages Of Jekyll
* ### Requires no database
All posts and pages in Jekyll are converted to static HTML prior to publication. This significantly reduces complexity that databases may introduce.
* ### Speed
Due to the lack of database calls when a page is loaded, the loading speed of the page is improved.
* ### Security
Vulnerabilities are greatly reduced as there is no CMS or database which requires frequent security updates.
* ### Free and Convenient hosting
Lastly, [Github Pages](https://pages.github.com/ "Github Pages") allows one user website to be hosted for free. Github directly builds and hosts the website. This means that the user doesn't even need to install Ruby on their local system.
